Courses Offered

  1. Computer System

    Learn about the fundamental components of computer systems, including hardware, software, and networking.

  2. Number System and Conservation Boolean Logic

    Explore different number systems and Boolean logic, essential for understanding computer operations and digital circuits.

  3. Computer Software and Operating System

    Dive into the world of software development and operating systems, learning how they manage computer resources.

  4. Application Packages

    Get hands-on experience with various application packages used for word processing, spreadsheets, and presentations.

  5. Programming Concepts and Logics

    Understand the fundamentals of programming, including syntax, control structures, and problem-solving techniques.

  6. Web Technology I

    Learn the basics of web development, including HTML, CSS, and JavaScript, to create dynamic and responsive websites.

  7. Multimedia

    Explore the creation and manipulation of multimedia content, including images, audio, and video.
